# Logitech G923 wheel baseline

Date declared: 2026-07-23
Decision: Q-006 Option A

## Declared device

- Wheel: Logitech G923 Racing Wheel
- Variant: Xbox Series X|S / Xbox One / PC
- Pedals: matching three-pedal set
- Force-feedback family: Logitech TRUEFORCE-capable hardware
- Owner availability: purchased November 2025
- Primary validation host: the declared Windows 11 reference PC

## Role

This physical device defines the first MVP wheel and pedal discovery,
calibration, mapping, disconnect/reconnect, and human handling baseline.
Selecting it does not claim that wheel support, force feedback, or TRUEFORCE is
implemented or approved.

Force feedback remains optional for the first wheel slice. Basic steering,
accelerator, brake, clutch, buttons, paddles, calibration, and safe input-state
behavior must pass before enhanced feedback can become a requirement.
