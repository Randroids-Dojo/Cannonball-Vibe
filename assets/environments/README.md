# Regional environment assets

P1-010 currently uses a project-original procedural Colorado proof-corridor kit.
The meshes and materials are generated at runtime by
`game/World/Environment/EnvironmentVisualKit.cs`; this directory is reserved for
future reviewed source art and optimized imports.

Do not add third-party terrain, vegetation, building, skyline, sky, or weather
assets without a rights record in `data/assets/environments/registry.json`.
